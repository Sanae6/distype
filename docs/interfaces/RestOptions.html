ratelimits

ratelimits: { globalPerSecond: number; pause: number; sweepInterval: number | false }
  • Defined in src/rest/RestOptions.ts:11

Ratelimit options.

Type declaration

  • globalPerSecond: number

    The amount of requests to allow to be sent per second. Note that this only applies to a single ClientWorker instance (If ClientMaster / ClientWorker are being used), meaning that you still may encounter 429 errors from global ratelimits.

  • pause: number

    The amount of time in milliseconds to wait between ratelimited requests in the same bucket.

  • sweepInterval: number | false

    An interval in milliseconds in which to sweep inactive buckets. False disables sweeping buckets automatically.
