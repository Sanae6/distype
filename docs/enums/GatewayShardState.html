CONNECTING

CONNECTING = 1
  • Defined in src/gateway/GatewayShard.ts:112

The shard is connecting. GatewayShard#_ws may be defined, however the connection process has not finished. During this stage, the GatewayShard:

  • Waits for an opcode 10 "hello" payload
  • Responds with a heartbeat
  • Waits for the first heartbeat ACK
  • Sends an identify payload
  • Waits for the ready event
