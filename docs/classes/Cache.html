Class Cache

The cache manager. Contains cached data, and handles dispatched gateway events to keep the cache up to date. Keep in mind that there are many caveats to the gateway, and that real-world cache data may not directly reflect your cache control options. It is recommended that you research intents and the caveats to the gateway, to make sure your bot receives sufficient data for your use case. Discord API Reference
